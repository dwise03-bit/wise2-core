🎯 FASTEST WAY TO GET STARTED
═══════════════════════════════

STEP 1: On Mac Terminal (or Termius)
──────────────────────────────────────
Copy this ONE command:

    ~/start-wise2-guide.sh

That's it! This will:
  ✓ Start a web server
  ✓ Show you your Mac's IP address
  ✓ List available files
  ✓ Tell you the URL to visit

STEP 2: On iPhone Safari
──────────────────────────
The script output will show:

    http://192.168.X.X:8080
                    ↑
    (Your Mac's IP - copy from Terminal)

1. Open Safari on iPhone
2. Paste the URL
3. Click index.html
4. View interactive Agentic OS guide!

STEP 3: Manage Files via Termius
──────────────────────────────────
1. Open Termius on iPhone
2. SSH to your Mac (use IP from step 1)
3. Command: cd ~/Desktop/wise2-agentic-os-guide
4. List files: ls -la
5. Read files: cat README.txt

❓ TROUBLESHOOTING
═════════════════════

Q: "Can't find start-wise2-guide.sh"
A: Run: ls ~/start-wise2-guide.sh
   If not there, run in Terminal:
   chmod +x ~/start-wise2-guide.sh

Q: "Port 8080 already in use"
A: Kill the old server: lsof -i :8080 | grep LISTEN
   Then: kill -9 [PID]
   Or use different port:
   python3 -m http.server 8081

Q: "Can't connect from iPhone"
A: • Make sure both on same WiFi
   • Check Mac IP: ipconfig getifaddr en0
   • Check firewall: System Preferences → Network
   • Try: http://localhost:8080 on Mac first

Q: "index.html looks wrong"
A: Make sure you're at port 8080
   Try: http://YOUR_IP:8080/index.html
   Not: http://YOUR_IP/index.html
