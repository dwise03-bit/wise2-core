## TikTok Strategy

### Platform Key Facts
- **Algorithm:** Favors watch time, shares, comments (not followers)
- **Sweet Spot:** 21-34 seconds (keeps people watching)
- **Hook Window:** First 1-3 seconds (or you lose them)
- **Sound:** Use trending sounds for 3x+ views
- **Captions:** Text overlay required (95% watched muted)
- **Calls-to-Action:** Link in bio, comment, share

### Viral Formula
1. **Thumb-stopping hook** (first 1 second)
2. **Problem identification** (seconds 2-5)
3. **Solution reveal** (seconds 6-15)
4. **Proof/results** (seconds 16-25)
5. **Strong CTA** (last 3 seconds)

## Production Notes for All Scripts

### Technical Requirements
- **Resolution:** 1080x1920px (9:16 vertical)
- **Frame Rate:** 30fps (TikTok standard)
- **Duration:** 15-30 seconds (optimal)
- **Format:** MP4, H.264
- **Sound:** AAC, mono or stereo
- **Captions:** Burned-in text (white, sans-serif)

### Filming Setup
- **Phone:** Landscape phone mounted on tripod
- **Lighting:** Natural light + key light (avoid shadows)
- **Background:** Clean desk setup
- **Camera:** iPhone/Android, TikTok app native
- **Audio:** Use TikTok's built-in sound library

### Post-Production Checklist
- [ ] Edit in TikTok app or CapCut (native TikTok compatibility)
- [ ] Add trending sound (pick THIS WEEK's trending audio)
- [ ] Add text overlays (white text, readable)
- [ ] Adjust color grading (bright, appealing)
- [ ] Add smooth transitions
- [ ] Export at correct resolution
- [ ] Test on mobile before posting

### Posting Strategy
**Optimal Times:**
- Tuesday-Thursday, 6-9 PM (peak teen/young adult viewing)
- Post same video 2-3 times per week (algorithm resets)
- Post to TikTok first, then cross-post to Instagram Reels + YouTube Shorts

**First 24 Hours are Critical:**
- Monitor comments (respond to top comments)
- Share with friends to get initial engagement
- Pin top comment to encourage responses
- Track engagement (views, shares, comments)

**Engagement Loop:**
- Respond to every comment (especially first hour)
- Reply with video responses to popular comments
- Create follow-up videos based on comments
- Pin top comments that spark discussion

### Caption Strategy
**Text Overlay Best Practices:**
- White text with dark outline (readable over any background)
- Large font size (readable at phone size)
- 2-3 words per line maximum
- Sync text with video beat
- Use emojis sparingly (1-2 per scene)

### Sound Strategy
**TikTok Sound Rules:**
- Always use trending sounds (3x+ reach than original audio)
- Pick sounds that match the vibe
- Check sound trending status (in-app)
- Alternate between different sounds across videos
- Avoid copyrighted music (unless royalty-free)
