# Wise² Core — Complete Brand System

**Master Index for All Brand Assets & Documentation**

---

## Overview

This directory contains the complete, comprehensive brand system for Wise² Core. Everything you need to consistently represent the brand across all touchpoints is here.

**Total Documentation:** 6 complete guides + 1 context file + foundational .agents files

---

## Files in This Directory

### 1. **01-BRAND-STRATEGY.md**
**What it is:** Full brand strategy report covering positioning, market analysis, competitive landscape, target audiences, and strategic vision.

**Who needs it:** Leadership, marketing strategists, anyone making strategic decisions about the brand.

**Key sections:**
- Brand vision and mission
- Target audience personas
- Competitive analysis
- Positioning statement
- Brand values and goals

**Length:** ~3,000 words | **Time to read:** 15–20 min

---

### 2. **02-BRAND-VOICE.md**
**What it is:** Complete verbal identity guide covering tone, vocabulary, writing style, and communication principles.

**Who needs it:** Copywriters, content creators, customer support, anyone writing on behalf of Wise² Core.

**Key sections:**
- Voice essence and tone dimensions
- 6 core voice qualities with do/don't examples
- Vocabulary (words we use vs. avoid)
- Writing style rules
- Before/after examples
- Things we never say

**Length:** ~2,500 words | **Time to read:** 12–15 min

---

### 3. **03-BRAND-MESSAGING.md**
**What it is:** Messaging framework including core message, value proposition, taglines, and key messages with proof points.

**Who needs it:** Marketing team, copywriters, sales team, campaign builders.

**Key sections:**
- Core message (the single most important thing)
- Value proposition
- 5 tagline options (with recommended choice)
- 4-level messaging hierarchy
- Audience-specific messaging
- Channel-specific adaptations

**Length:** ~2,200 words | **Time to read:** 10–12 min

---

### 4. **04-BRAND-IDENTITY.md**
**What it is:** Visual identity brief covering logo, colors, typography, imagery, icons, and design principles.

**Who needs it:** Designers, product designers, marketing designers, creative directors.

**Key sections:**
- Logo direction and usage rules
- Complete color palette with hex codes and usage rules
- Typography system (primary + secondary fonts)
- Photography and illustration style
- 4 core design principles
- Brand expressions (how to apply on different touchpoints)

**Length:** ~2,800 words | **Time to read:** 15–18 min

---

### 5. **05-D2C-MARKETING.md**
**What it is:** Complete direct-to-founder marketing strategy covering acquisition, retention, expansion, and growth tactics.

**Who needs it:** Marketing team, growth strategists, product strategists, community managers.

**Key sections:**
- Unit economics baseline
- Acquisition channel strategy
- Owned channel strategy (email + community)
- Retention and expansion mechanics
- Social proof system
- Content strategy
- Launch checklist
- Metrics dashboard

**Length:** ~2,400 words | **Time to read:** 12–15 min

---

### 6. **06-BRAND-GUIDELINES.md**
**What it is:** Master reference document that synthesizes all brand work into one comprehensive, actionable guide.

**Who needs it:** Everyone at Wise² Core. This is the single source of truth.

**Key sections:**
- Brand essence (mission, vision, values, personality)
- Visual identity quick reference
- Verbal identity quick reference
- Messaging quick reference
- Brand in use (digital, print, social)
- Critical brand don'ts
- Pre-publication checklist

**Length:** ~4,000 words | **Time to read:** 20–25 min (full reference); can be skimmed for specific sections

## How to Use This System

### For Designers:
1. Start with **04-BRAND-IDENTITY.md** for visual direction
2. Reference **06-BRAND-GUIDELINES.md** for quick visual rules
3. Refer to **01-BRAND-STRATEGY.md** for strategic context when making design decisions

### For Copywriters:
1. Start with **02-BRAND-VOICE.md** for tone and vocabulary
2. Use **03-BRAND-MESSAGING.md** for core messages and proof points
3. Check **06-BRAND-GUIDELINES.md** before publishing anything

### For Marketing:
1. Start with **01-BRAND-STRATEGY.md** for positioning and audience
2. Use **03-BRAND-MESSAGING.md** for campaign messaging
3. Follow **05-D2C-MARKETING.md** for launch and growth plans
4. Reference **06-BRAND-GUIDELINES.md** for visual and verbal consistency

### For Product/Engineering:
1. Read **01-BRAND-STRATEGY.md** to understand the vision and audience
2. Reference **04-BRAND-IDENTITY.md** for UI/design system choices
3. Use **02-BRAND-VOICE.md** for in-app copy and error messages

### For Leadership:
1. Start with **01-BRAND-STRATEGY.md** for full strategic context
2. Skim **06-BRAND-GUIDELINES.md** for quick reference
3. Use **05-D2C-MARKETING.md** to understand go-to-market

## Quick Reference Card

### Core Message
**Wise² Core helps ambitious business owners scale their product to millions without hiring a DevOps team or doubling their budget.**

### Tagline (Recommended)
**"Build like you have a DevOps team. Pay like you don't."**

### Brand Personality
**Hustler · Practical · Action-oriented · Reliable · Ambitious**

### Brand Colors
- **Primary:** Navy (#1A1F36) + Action Orange (#FF6B35)
- **Secondary:** Gray (#6B7280), Light (#F9FAFB), Green (#10B981), Red (#EF4444)

### Brand Fonts
- **Primary:** Inter
- **Code:** JetBrains Mono

### Brand Voice
Trusted peer builder—honest, practical, grounded in execution, unwilling to waste your time.

### Do/Don't Checklist (Before Publishing)
- [ ] Honest and grounded (not overpromising)?
- [ ] Clear value or benefit?
- [ ] Follows visual system?
- [ ] Active voice with specific claims?
- [ ] Sounds like someone who ships?
- [ ] Using on-brand vocabulary?
- [ ] Conversational and direct tone?
- [ ] Clear, ambitious promise?
