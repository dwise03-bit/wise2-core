## How to Use This Vault

### For recording decisions:
- Use `/obsidian-save` after meetings/sessions
- Every decision gets its own note with [[wikilinks]] to related items
- Contradictions are auto-detected and flagged

### For synthesis:
- Use `/obsidian-synthesize` to find unnamed patterns
- Use `/obsidian-reconcile` to resolve conflicting claims
- Use `/obsidian-review` for periodic audits

### For research:
- Use `/research` to capture web research
- Use `/research-deep` for thorough investigation
- Use `/youtube` or `/podcast` for media ingestion
- Everything saves to vault with source attribution

### For scheduling:
- Use `/obsidian-task` to create tracked tasks
- Use `/obsidian-recurring` for repeating work
- Use scheduled agents for nightly/weekly automation
